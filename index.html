Política de Privacidade

Última atualização: maio de 2026

Resumo simples: o Caixinha coleta apenas o necessário para o app funcionar. Não vendemos seus dados, não exibimos anúncios baseados em perfil e não compartilhamos suas informações com terceiros além dos serviços essenciais de funcionamento do app (Firebase e RevenueCat).

1 Quem somos

O Caixinha é um aplicativo de gestão financeira doméstica desenvolvido por Iago Mollon Esqueisaro, desenvolvedor independente. Para dúvidas ou solicitações relacionadas à privacidade, entre em contato pelo e-mail iagometech@gmail.com.

2 Dados que coletamos

O Caixinha coleta apenas os dados necessários para o funcionamento do aplicativo:

  • Nome e endereço de e-mail — obtidos no momento do cadastro ou via autenticação com Google;
  • Foto de perfil — obtida via autenticação com Google, quando aplicável;
  • Senha — quando o usuário opta pelo login com e-mail e senha. As senhas são gerenciadas exclusivamente pelo Firebase Authentication e nunca são armazenadas ou acessadas em texto simples por nós;
  • Dados financeiros domésticos — despesas, categorias, valores e datas cadastrados pelo próprio usuário dentro do app;
  • Dados de lares e membros — nome do lar e identificadores dos usuários membros, necessários para a funcionalidade de gestão compartilhada.
Não coletamos dados de localização, contatos, câmera, microfone ou qualquer outro dado sensível do dispositivo.

3 Como usamos seus dados

Os dados coletados são utilizados exclusivamente para:

  • Autenticar o usuário e manter a sessão ativa no app;
  • Sincronizar despesas e lista de compras entre os membros do mesmo lar em tempo real;
  • Exibir informações financeiras do lar (dashboard, saldo mensal, histórico);
  • Gerenciar a assinatura premium do usuário.

Não utilizamos seus dados para fins publicitários, análise de perfil comportamental ou qualquer finalidade além das descritas acima.

4 Compartilhamento de dados

Seus dados são compartilhados apenas com os seguintes serviços essenciais para o funcionamento do app:

  • Google Firebase (Firestore e Authentication) — armazenamento de dados e autenticação. Política de privacidade: firebase.google.com/support/privacy
  • RevenueCat — gerenciamento de assinaturas premium. Política de privacidade: revenuecat.com/privacy

Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins comerciais ou publicitários.

5 Dados compartilhados dentro do lar

O Caixinha é um app colaborativo. Ao criar ou participar de um lar, as despesas que você lançar, os itens que adicionar à lista de compras e seu nome serão visíveis para os demais membros do mesmo lar. Esse compartilhamento é intencional e essencial para o funcionamento do app.

Dados de lares diferentes são completamente isolados entre si.

6 Retenção de dados

Seus dados são mantidos enquanto sua conta estiver ativa. Ao excluir sua conta, todos os seus dados pessoais são removidos permanentemente, incluindo:

  • Lares dos quais você é o único membro (excluídos integralmente);
  • Lares com outros membros: a propriedade é transferida automaticamente para outro membro antes da exclusão;
  • Todos os demais dados associados à sua conta (despesas, preferências, dados de assinatura).

A exclusão pode ser realizada diretamente na tela de perfil do aplicativo.

7 Segurança

Todos os dados transmitidos entre o app e nossos servidores são criptografados em trânsito via HTTPS/TLS. As senhas são gerenciadas pelo Firebase Authentication e nunca ficam acessíveis em texto simples. O acesso aos dados do Firestore é protegido por regras de segurança que garantem que cada usuário acesse apenas os dados dos lares dos quais faz parte.

8 Seus direitos

Você tem direito a:

  • Acessar os dados que o app armazena sobre você;
  • Corrigir dados incorretos diretamente no app;
  • Excluir sua conta e todos os dados associados a qualquer momento, pela tela de perfil;
  • Solicitar informações sobre o tratamento dos seus dados entrando em contato pelo e-mail abaixo.

Esta política está em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

9 Menores de idade

O Caixinha não é direcionado a menores de 13 anos e não coleta intencionalmente dados de crianças. Caso identifiquemos o cadastro de um menor, a conta será removida.

10 Alterações nesta política

Esta política pode ser atualizada ocasionalmente. Em caso de alterações relevantes, notificaremos os usuários dentro do próprio aplicativo. A data da última atualização sempre será exibida no topo desta página.
