# 🏠 Caixinha — Política de Privacidade

Este repositório hospeda a política de privacidade do aplicativo **Caixinha**, disponível na Google Play Store.

## Sobre o Caixinha

Caixinha é um app de gestão financeira doméstica para Android. Permite que famílias e grupos controlem gastos compartilhados, gerenciem listas de compras em tempo real e acompanhem o orçamento mensal do lar.

📲 **Download na Play Store:**
[play.google.com/store/apps/details?id=com.iagome.caixinha](https://play.google.com/store/apps/details?id=com.iagome.caixinha)
